**Support - OpenRouter Usage Monitor**

Live usage and billing tracking for your OpenRouter account in your menu bar.

**Getting Started**

1. Launch OpenRouter Usage Monitor
2. Click the menu bar icon and select "Settings"
3. Visit [openrouter.ai/settings/management-keys](https://openrouter.ai/settings/management-keys)
4. Create a new Management API key (not a regular API key)
5. Copy and paste the key into the app's settings
6. Click "Save Key"

**Features**

- Real-time credit balance monitoring
- Usage tracking by model and time period
- Multiple menu bar display modes:
  - Icon only: Minimal display
  - Balance: Shows current credit balance
  - Usage: Shows credit usage for selected period
  - Both: Shows both balance and usage
- Automatic refresh every 5 minutes
- Secure key storage in Keychain

**Menu Bar Display Options**

Configure what information appears in your menu bar:
- Click the menu bar icon → Settings → Menu Bar section
- Choose from 4 display modes
- Preview shown in settings before applying

**Common Issues**

*API Key Not Working*
- Ensure you're using a Management API key, not a regular API key
- Management keys are found at openrouter.ai/settings/management-keys
- Regular API keys (for making requests) won't work with this app

*Data Not Updating*
- The app refreshes every 5 minutes automatically
- Manually refresh by clicking the menu bar icon → Refresh
- Check your internet connection
- Verify your API key is still valid

*Menu Bar Icon Not Appearing*
- Check System Settings → General → Login Items & Extensions
- Ensure OpenRouter Usage Monitor is allowed to run in the background
- Try restarting the app

**System Requirements**

- macOS 15.2 or later
- Internet connection for API access
- OpenRouter account with Management API key

**Privacy & Security**

- Your API key is stored securely in the Keychain
- All data is fetched directly from OpenRouter's API
- No data is collected or transmitted to third parties
- See our [Privacy Policy](https://kandelvijaya.github.io/Terms/PrivacyPolicy_OpenRouterUsageMonitor)
